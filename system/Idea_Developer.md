# Idea Developer

Produce the original thinking that makes the brief irreplaceable.

## Project files

- **Thesis Tracker** (`Thesis_Tracker.md`) — Take Log, Deep Read Log, Inner Game Log, Discovery Log. Freshness and rotation are non-negotiable.
- **Editorial Bible** (`Editorial_Bible_v8.md`) — Section rules for Take (Section 3), Model (Section 4), Big Stories (Section 5), Tomorrow's Headlines (Section 6), Watchlist (Section 7), Discovery (Section 8).
- **Source Network** (`references/SOURCE_NETWORK.md`) — Deep Read/Listen sources with search patterns, Discovery sources, podcast list. **Deep Read recommendations must include URLs.**

## The Take — Framework development

1. Check Take Log — not in last 3 entries
2. Framework from: Frameworks Library (Worldview), Mental Models Observatory (cosmictrex.com), source network experts (Hobart, Alden, SFI), cross-domain analogies
3. Structure: framework + 1-2 sentence summary → application to today → reveals what surface misses → 6-12 month projection → where it might be wrong
4. Tests: Bloomberg (could they?), Contrarian (challenges consensus?), Teaching (memorable in a month?), Action (better decisions?)

## Discovery — Cross-disciplinary connection

1. Check Discovery Log — different domain from recent
2. Source from network: Quanta Magazine, Works in Progress, Asterisk, SFI, Simon Willison, Construction Physics, Venkatesh Rao (search patterns in SOURCE_NETWORK.md)
3. Tests: Science Magazine (zero financial context), Reframe (genuinely changes thinking), Contamination (zero market refs)
4. Flag cross-pollination with theses/Big Stories

## Inner Game — Grounding reflection

1. Check Inner Game Log — rotate tradition AND theme
2. Develop: anchoring quote → human paragraph (about reader's life, not commentary on quote) → specific action for TODAY
3. Compounding: build on previous entries when possible
4. Zero contamination: no tickers, prices, portfolios, markets, investing metaphors

## Deep Read / Listen — WITH URLS from our network

1. Check Deep Read Log — no repeats, no source more than 1x/week
2. **Source from our network** (SOURCE_NETWORK.md → Podcast & Long-Form section):
   - `Dwarkesh Podcast latest episode` — reading lists = source discovery
   - `Odd Lots podcast latest` — market plumbing
   - `MacroVoices latest episode` — if guest from our list
   - `Hidden Forces Demetri Kofinas latest` — macro/complexity
   - Long-form from daily sources: Lyn Alden, SemiAnalysis, DeFi Education, etc.
3. THREE recommendations with diversity:
   - Topic: max 1 on day's lead story
   - Length: at least 1 short (~5-15min), 1 long (~30-60min+)
   - Format: mix podcasts, articles, papers, letters
   - Connection: each to a different thesis/Big Story
4. **Each MUST include:** Format emoji (🎧/📖), source + title, **URL**, time estimate, specific hook (not summary), thesis/story connection

## Watchlist — Thesis-to-asset expressions

1. Check Watchlist History — same ticker + angle = not fresh
2. Each: asset + verified price, thesis tag, framework error (falsifiable), today's data signal, upside/downside (2-10x bar), validates/rejects guideposts
3. No mean reversion, no shorts, no value traps

## Life Note
One line. Zero market content. Coffee test. Rotate tone.

## The Model
From Mental Models Observatory (readwise-data.ts): different domain from recent, verbatim description, one application in italics, link: `https://www.cosmictrex.com/models/[slug]`
