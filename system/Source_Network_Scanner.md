# Source Network Scanner

Gather today's signal from our curated source network. These sources surface insight 2-4 weeks before mainstream coverage. They are the brief's competitive advantage.

## Project files

- **Worldview** (`Current_Worldview_v5.md`) — active theses, Big Stories, Tomorrow's Headlines. These GUIDE what you're looking for.
- **Thesis Tracker** (`Thesis_Tracker.md`) — rotation logs (Deep Read, Take, Discovery, Inner Game) to identify recommendation candidates and avoid repeats.
- **Source Network** (`references/SOURCE_NETWORK.md`) — every source with search patterns, URLs, signal types, and thesis/story links. **This is the primary operational reference for this skill.**

## Core principle

**Check our sources FIRST. Google News is the safety net, not the starting point.** If the brief's best insights come from the general sweep, the source network has a gap.

## Phase 1: Daily must-checks (~15 searches)

Run these search patterns from SOURCE_NETWORK.md every session:

**Macro:**
- `Byrne Hobart The Diff latest` → what he covers = what we need a view on
- `Lyn Alden latest article` → fiscal dominance, structural macro (Thesis 2, 5; BS #5, #19, #20)
- `Luke Gromen FFTT latest` → sovereign debt, gold/BTC reserve (Thesis 5; BS #4, #20)
- `Pine Analytics substack latest` → macro → crypto bridge

**Crypto:**
- `site:dune.com/hildobby` → ETF flows. CRITICAL daily data. (BS #3; Thesis 3)
- `Noelle Acheson crypto is macro now latest` → structural crypto-macro
- `DeFi Education substack latest` → protocol fundamentals

**AI & Tech:**
- `Simon Willison latest` → 2-4 weeks ahead of mainstream. Discovery candidates originate here.
- `SemiAnalysis latest` → AI hardware, data centers (BS #7; Thesis 4)
- `Latent Space podcast latest` → AI engineering, source discovery via guest network

**Cross-domain:**
- `Quanta Magazine latest` → Discovery primary source
- `Works in Progress magazine latest` → Discovery + Deep Read candidate

### Phase 5: General news sweep (LAST — safety net only)

Only after checking our network:
- `major financial news today [date]`
- `[specific event/country] latest` for stories without dedicated sources
- `earnings reports today` / `economic data releases today`
- `AI news today` — catch company announcements

## Routing each finding

Tag every Tier 1/2 finding: → Big Story #N / → Tomorrow's Headlines #N / → The Six [subsection] / → Take candidate / → Discovery candidate / → Watchlist signal / → Deep Read candidate (with URL, format, time, hook)

Also tag: which thesis it strengthens/weakens, which source surfaced it.

## Cross-reference against accumulated knowledge

Before finalizing, check findings against: Big Stories (updated/silent?), Tomorrow's Headlines (greenshoots?), active theses (strengthen/weaken?), Take Log (overlap?), Watchlist History (new expression?), Deep Read Log (no repeats?).

## Output

Source Network Report with: network signal by domain (with URLs), tiered findings with routing, Big Story status, thesis evidence, Deep Read candidates with URLs/hooks, source network notes (unusual signal, silence, new voices), gaps filled by Phase 5.
